Detailed Description

template<typename T>
class tensor4d::nn::Linear< T >

Linear (Dense/Fully Connected) layer.

Performs: output = input @ weights^T + bias

Uses GPU if available and compiled with USE_GPU, otherwise falls back to BLAS or CPU. Backend selection follows: GPU → BLAS → CPU
