Detailed Description

template<typename T, size_t N>
class Adam< T, N >

Adam (Adaptive Moment Estimation) optimizer.

Combines ideas from RMSprop and momentum. Maintains running averages of both gradients and their squares.

Update rule: m_t = beta1 * m_{t-1} + (1 - beta1) * grad v_t = beta2 * v_{t-1} + (1 - beta2) * grad^2 m_hat = m_t / (1 - beta1^t) v_hat = v_t / (1 - beta2^t) param = param - lr * m_hat / (sqrt(v_hat) + epsilon)
