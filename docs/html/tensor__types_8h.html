Detailed Description

Type aliases for common Tensor specializations.

This header provides convenient type aliases for commonly used Vector and Matrix types. These are pre-instantiated in the library for faster compilation and smaller binary size.

Usage Example

#include "tensor_types.h"
using namespace tensor4d;
// Create a float vector
Vectorf v({10});
v.fill(1.0f);
// Create a double matrix
Matrixd M({3, 3});
M.fill(0.0);
// Create a 3D tensor
Tensor3f data({4, 5, 6}); // 4 channels, 5 rows, 6 cols
// All aliases support the full tensor API:
auto result = v.exp().sum();
auto product = M.matmul(M);
