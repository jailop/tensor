Detailed Description

template<typename T>
class MemoryPool< T >

Thread-safe memory pool for efficient tensor allocation.

Reduces allocation overhead by reusing previously allocated memory blocks. Useful for repeated tensor operations with similar sizes.

Template Parameters
TData type to allocate
MemoryPool<float> pool;
auto ptr = pool.allocate(1000); // Allocate 1000 floats
// ... use memory ...
pool.deallocate(ptr, 1000); // Return to pool for reuse
